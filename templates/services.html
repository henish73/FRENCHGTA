Course Structure by Level

Beginner French (A1–A2)

Learn foundational grammar, sentence structures, and basic conversation. Perfect for absolute beginners.

  • Course Duration: 3–4 months (with 5 classes/week)
  • Class Time: 60 minutes per class
  • Learning Goals: Introductions, daily routine, directions, food, numbers

Intermediate French (B1–B2)

Build conversational fluency. Master tenses, opinions, and professional dialogues.

  • Course Duration: 4–5 months (with 5 classes/week)
  • Class Time: 60 minutes per class
  • Learning Goals: Express ideas clearly, discuss topics like work, travel, health

Advanced French (C1–C2)

Become near-native in French. Understand idioms, culture, formal writing, and fast native speech.

  • Course Duration: 5–6 months (with 5 classes/week)
  • Class Time: 60 minutes per class
  • Learning Goals: Fluent debates, literature, academic essays, interview prep

TEF Canada Preparation

Focused test preparation for all TEF sections: reading, listening, speaking, and writing. We guide you with mock tests and scoring strategies.

  • Recommended Duration: 2–3 months (depending on target score)
  • Class Time: 60 minutes per session
  • Includes: Section-wise practice, scoring rubric, weekly tests
